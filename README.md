# Programming Joke App

## Setup Instructions

1. Clone the repository:
   git clone <repo-url>

2. Navigate into the project:
   cd joke-app

3. Install dependencies:
   npm install

4. Start development server:
   npm run dev

## Usage

- A programming joke loads automatically when the app starts.
- Click "Get a New Joke" to fetch a new joke.
- A loading message is displayed while fetching.

## Technologies Used

- React
- useState
- useEffect
- JokeAPI
